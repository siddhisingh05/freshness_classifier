Computer Vision for Food Safety FreshVision

FreshVision is a three-class image classifier that determines whether produce is fresh or spoiled. Upload a photo and the model returns a confidence score for each class in under a second.

Preprocessing

Uploaded images are sent directly to the FreshVision inference engine. The model handles all internal preprocessing including resizing and normalization before classification.

Inference

FreshVision uses a deep learning model hosted on Hugging Face, accessed via the Gradio API. Each image is analysed and returns a freshness score out of 100 along with a classification label.

Tech Stack
  • Backend: Flask 3.0, Python 3.10, Gunicorn
  • ML: TensorFlow / Keras, NumPy
  • Image: Pillow
  • Database: SQLite
  • Auth: Flask-Bcrypt
  • Deploy: Docker, docker-compose
